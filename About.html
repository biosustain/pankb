About PanKB


Welcome to PanKB, the Pangenome Knowledge Base. This web-based tool serves as an up-to-date resource for pangenomics research methods and the dissemination of results. The current version of PanKB offers basic pangenome characteristics and advanced pangenome analytics for six major biotech bacteria families.


How to use PanKB

To begin, choose an organism of interest from the organism list. You can access the organism list page by either clicking the 'Select an organism' button located at the bottom of the landing page or selecting the 'Organism' option in the navbar. Next, you can pick an organism of interest and explore its pangenomic analytics by clicking on the link in the last column labeled 'Pangenome analyses.' This action will direct you to the Pangenome Analysis Dashboards.

Alternatively, you can access the overview page of a specific species by clicking on the data point within the 'Pangenomes' figure located in the middle of the landing page.



Pangenome Analytics Dashboards


Each dashboard showcases the pangenome analysis results for a organism species. It consists of three parts: Overview, Genes, and Phylogenetic Tree. In the following section, we provide examples of each page element from the Lactiplantibacillus plantarum Pangenome Analytics Dashboards.

By default, when you enter an organism's Pangenome Analytics Dashboards, you will first see the overview page. You can navigate to the other two pages using the sidebar on the left.

Overview


The Overview page contains several interactive figures that depict various pangenome properties of an organism.

Heap's Law and Gene Frequency


The first 4 figures in the Overview page display the four standard calculations related to elementary pangenome properties: Heap's Law, Cumulative Gene Distribution, Gene Annotation Distribution, and Gene Frequency. All of them are interactive figures that offer the following functionalities:
1. Hovering: By using the mouse to hover over a figure element (line, bar, etc.), you can access more details about that particular element.
2. Hide and show: Clicking on the legends allows you to hide or show the corresponding Gene class (Core, Accessory, Rare) in the figure.
3.Zoom in and Zoom out: If you wish to zoom in on a specific area of a figure, simply press and hold the left mouse button within the figure while selecting the corresponding area. To undo the zoom in, just double-click the left mouse button.



Presence/Absence Matrix


The Presence/Absence Matrix is represented as an interactive heatmap. By default, the heatmap displays the Presence/Absence Matrix only for Core genes. To explore other gene classes such as Accessory, Rare, etc., you can use the 'Select a Gene Class' button located in the upper left corner of the figure.

As an interactive heatmap, the Presence/Absence matrix has the following features:
1. Zooming In and Out: You can zoom in and out on the x-axis and y-axis of the heatmap by dragging the two sliders located in the upper left corner of the heatmap. The horizontal and vertical lines surrounding the small lock icon correspond to these axes. To simultaneously zoom in on both the x-axis and y-axis, simply click on the small lock icon and then drag either of the two lines.
2. Hover Functionality: Hover your mouse over a cell in the heatmap to view information related to the corresponding gene (x-axis label) and genome (y-axis label).
3. Links to Other Pages: Click on a cell of interest to navigate to the genome or gene page that corresponds to that cell. On these pages, you can access more detailed information about the genome and gene (refer to the following section: Genome and Gene Info Page).



Genome & Gene Info Page


This page offers more information about the genome and genes from the Presence/Absence Matrix. The page mainly consists of three parts:
1. Genome and Gene Tables: These two tables contain basic information about the genome and genes. The genome table includes a link to NCBI, where you can find more information about this genome. The gene table also provides a link to the Gene page (See Gene Page).
2. Gene Annotation Distribution: This is a similar interactive bar plot to the Gene Annotation Distribution on the Overview page. However, this plot focuses on a single genome, whereas the one on the Overview page covers the entire species.
3. Secondary Metabolite BGCs: This section displays the results generated by antiSMASH. If you would like to learn more about antiSMASH, please visit antiSMASH.


Genes

Genes page mainly show the annotation of genes.

Gene Annotation Table


The Gene Annotation table is an interactive table on the Genes page. It possesses the following features:
1.Sorting: You can sort the entire table based on any single column.
2.Searching: You can enter filter conditions in the search box of each column to precisely filter the data you desire.
3.Links to other pages: Each entry in the first column of the Gene Annotation Table serves not only as the gene name but also as a link to the corresponding Gene Info Page. These gene pages contain more comprehensive gene information and advanced pangenome analyses.


Gene Info Page


The Gene page currently comprises two components:
1. Gene Information Table: This interactive table contains all Locustags related to this gene, along with their corresponding Genome ID (found in the second column labeled 'Genome ID' which also serves as a link to the respective Genome Info Page), annotation details, and sequence information.
2. Multiple Sequence Alignment: Each core gene is accompanied by a Multiple Sequence Alignment. This alignment is presented as an interactive figure with the following features:
        1.Overview Box: Located at the bottom of the figure, the overview box allows you to navigate the multiple sequence alignment plot. You can also select a specific area on the overview box using the left mouse button, and the multiple sequence alignment will jump to the selected region.
        2.Scrolling Function: You can scroll up, down, left, and right by holding down the left mouse button and dragging the image.
        3.Amino Acid Percentage: Clicking on the position number (found below the logo plot) enables you to view the amino acid percentage for each position.
        4.Links to Other Pages: You can access the corresponding Genome's Genome Info Page by clicking on the Genome ID along the y-axis. The Genome Info Page is similar to the Genome and Gene Information Page but does not include the gene table (refer to the above section for the link).


Phylogenetic Tree

The Phylogenetic Tree page displays the phylogenetic traits of a species, and it is constructed using the Mash distance method. The phylogenetic tree in PanKB is generated using Phylotree.js. You can find additional information about its features in phylotree.js. In addition to the features documented in the Phylotree.js documentation, we have also incorporated links to the Genome Info Page. By left-clicking on the genome of interest and selecting 'Search for this strain,' you can navigate to the corresponding Genome Info Page and access more information about that specific genome.
